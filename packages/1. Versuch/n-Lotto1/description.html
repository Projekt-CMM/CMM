Lottozahlen (Teil 1)

Einführung

Beim Lotto 6 aus 45 werden insgesamt 6 Kugeln mit jeweils einer Zahl zwischen 1 und 45 gezogen. Eine Ziehung besteht also aus 6 Zahlen.


Aufgabe

Simuliere eine Lotto-Ziehung. Die zufällig generierten Zahlen sollen in ein Array gespeichert und mit dem Befehl printIntArray ausgegeben werden (siehe Hinweise). Eine Zahl darf mehrmals vorkommen.

Wichtig

Der Zufallsgenerator muss wie folgt initialisiert werden:

srand(atoi(scanf()));
Als Eingabedaten kannst du eine beliebige ganze Zahl verwenden, zum Beispiel:
234543, 43567543, 435654, ...

Hinweis

Ein Array kann mit dem Befehl printIntArray ausgegeben werden. Dabei ist a das Array und n die Anzahl der Elemente des Arrays.

int a[n]; printIntArray(a, n);
Für diesen Befehl muss die Bibliothek array.h eingebunden werden.
#include <array.h>
