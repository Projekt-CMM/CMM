Array-10

Aufgabe

Schreibe ein Programm, das ein Array von 100 ganzen Zahlen anlegt, die sich als Produkt des Index mit 10 ergeben. Gib das Feld mit jeweils 10 Elementen Pro Zeile aus.

Beispiel:
array[43] = 430;

Hinweis

Ein Array kann mit dem Befehl printArray ausgegeben werden. Dabei ist a das Array und n die Anzahl der Elemente des Arrays.

int a[n]; printArray(a, n);
Für diesen Befehl muss die Bibliothek array.h eingebunden werden.
#include <array.h>
