Messwerte

Aufgabe

Erstelle ein C-Programm, das vom Benutzer bis zu 30 Messwerte (Gleitpunktzahlen) einlesen kann. Im Anschluss sollen jeweils der größte und der kleinste Wert und der arithmetische Mittelwert dieser Messreihe ausgegeben werden.
Format der Ausgabe:

Kleinste: 0.01 Groesste: 10.34 Durchschnitt: 4.42

Hinweise

In C Compact gibt es den Datentyp double nicht. Verwende float für die Messwerte.
Den arithmetischen Mittelwert (Durchschnitt) einer Zahlenmenge erhälst du, indem du alle Elemente addierst und das Ergebnis dann duch die Anzahl der Elemente (n) dividierst:


Eine Zahl kann dem Befehl readFloatArray eingelesen werden. Dabei ist a das Array und n die Anzahl der Elemente des Arrays.
readFloatArray(a, n);
Für diesen Befehl muss die Bibliothek array.h eingebunden werden.
#include <array.h>
